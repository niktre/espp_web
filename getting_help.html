Getting Help

The fastest way [GitHub repository]

  1. Go to the repository
  2. click the tab Issues
  3. check for active issues. If you don't find yours go to 4.
  4. click on New issue and raise a new one.

A more general way

You can subscribe to two ESPResSo++ mailing lists that provide help: for users and for developers
